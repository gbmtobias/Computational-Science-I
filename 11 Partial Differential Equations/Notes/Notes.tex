\section*{Exercise 1}

The following code solves the Vampire equation for the initial condition:

\begin{equation*}
f(x)=\begin{cases}0&\text{ if $x < 0.4$}\\\sin(x)&\text{ if 0.4 $\leq x\leq 0.6$}\\0&\text{ if $x > 0.6$}\end{cases}
\end{equation*}

The timestep is $10^{-5}$.

\lstinputlisting[language=Python]{../VampireEquation.py}

\section*{Exercise 2}

This codes solves the Schroedinger equation for the potential $V(x) = \frac{1}{2}(x-1)^{2}$. The initial condition is the square root of a gaussian $N(-4,1)$ distribution.

\lstinputlisting[language=Python]{../SchroedingerEquation.py}

The code generates an animated graph for the function $\psi$ (top plot) as well as the Fourier Transform $\tilde{\psi}$ (bottom plot). For both, $\psi$ and $\tilde{\psi}$, the real component (blue) and the imaginary component (green) are plotted. In the top plot the potential $V(x)$ is also plotted (red plot).

\end{center}
